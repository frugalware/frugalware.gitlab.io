A vulnerability has been reported in KVM, which can be exploited by malicious, local users to bypass certain security restrictions or cause a DoS (Denial of Service).
 The error can be exploited by a guest to read arbitrary files on the host via a specially crafted disk header. For more information, see FSA455.
