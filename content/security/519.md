* Vulnerable: 3.42-1
* Unaffected: 3.42_3-1kalgan1


A security issue has been reported in HTTrack, which potentially can be exploited by malicious people to compromise a vulnerable system.
 The security issue is caused due to a boundary error when processing command line arguments. This can be exploited to cause a buffer overflow by passing overly long URLs to the application.
* Bug Tracker URL: http://bugs.frugalware.org/task/3304
