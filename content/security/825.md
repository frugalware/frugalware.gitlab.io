* Author: kikadf
* Vulnerable: 3.9.5-1
* Unaffected: 3.9.5-2arcturus1


It was discovered that LibTIFF incorrectly handled certain malformed images when using the gif2tiff tool.
 If a user or automated system were tricked into opening a specially crafted GIF image, a remote attacker could crash the application, leading to a denial of service, or possibly execute arbitrary code with user privileges.
