A vulnerability has been reported in JasPer, which can be exploited by malicious people to cause a DoS (Denial of Service).
 The vulnerability is caused due to an error in the "jpc\_qcx\_getcompparms" function when processing JP2 files and can be exploited to crash an application using the library.
