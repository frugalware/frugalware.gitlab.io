A vulnerability has been reported in Drupal, which can be exploited by malicious people to bypass certain security restrictions.
 The vulnerability is caused due to the node\_access system restrictions not being enforced and can be exploited to view otherwise restricted nodes.
 NOTE: This affects the taxonomy and forum subsystems.
* Bug Tracker URL: http://bugs.frugalware.org/task/4521
