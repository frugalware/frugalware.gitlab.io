A vulnerability has been reported in Mozilla Firefox, which can be exploited by malicious people to compromise a user's system.
 The vulnerability is caused due to a use-after-free error when moving DOM nodes between documents and can be exploited via a specially crafted web page.
 Successful exploitation allows execution of arbitrary code.
* Bug Tracker URL: http://bugs.frugalware.org/task/4171
