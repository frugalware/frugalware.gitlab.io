Adhemerval Zanella Netto discovered that the GNU C Library incorrectly handled certain multibyte characters when using the iconv function.
 Tim Waugh discovered that the GNU C Library incorrectly enforced the WRDE\_NOCMD flag when handling the wordexp function.
