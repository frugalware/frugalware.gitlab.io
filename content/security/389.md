A weakness has been reported in KDE, which can be exploited by malicious, local users to cause a DoS (Denial of Service).
 The weakness is caused due to an error in KDM when processing certain files, which can be exploited to cause KDM to e.g. consume large amounts of memory or prevent users from logging in.
* Bug Tracker URL: http://bugs.frugalware.org/task/2681
