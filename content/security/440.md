* Vulnerable: 0.7.9-1
* Unaffected: 0.7.9-2kalgan1


A vulnerability has been reported in frugalwareutils, which can potentially be exploited by malicious people to cause a DoS on a vulnerable system.
 The vulnerability is caused due to creating new files as root without checking the current value of umask.
 Successful exploitation may allow execution of arbitrary code.
* Bug Tracker URL: http://bugs.frugalware.org/task/3052
## CVEs:
* There is no CVE for this issue.
