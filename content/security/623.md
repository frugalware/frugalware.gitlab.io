A vulnerability has been reported in the Comment RSS module for Drupal, which can be exploited to disclose potentially sensitive information.
 The vulnerability is caused due to the module not properly respecting access restrictions when adding the link to a node, which can be exploited to disclose potentially sensitive information.
* Bug Tracker URL: http://bugs.frugalware.org/task/3949

* No CVE references, see http://drupal.org/node/579280.
