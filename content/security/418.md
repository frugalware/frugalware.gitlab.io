David Leadbeater has reported a vulnerability in mtr, which can be exploited by malicious people to cause a DoS (Denial of Service).
 The vulnerability is caused due to mtr not properly limiting the length of some buffers.
* Bug Tracker URL: http://bugs.frugalware.org/task/2956

* There is no CVE for this issue.
