El Equipo de Desarrolladore de Frugalware tiene el placer de [anunciar](/news/160) la disponiblidad de Frugalware 1.2rc1, the first candidate of the upcoming 1.2 stable release.  

 A short and incomplete list of changes since 1.2pre2:  

* Improvements:
	+ We finally upgraded our KDE (and related) packages to the 4.x branch. The work started back in April 2009 and now we thought it's a good move to merge that testing repository.
	+ Updated artwork.
* Paquetes actualizados:
	+ KDE 4.3.5
	+ GNU C Library 2.11.1
	+ Mozilla Firefox 3.6
	+ Thunderbird 3.0.1
	+ More than 400 other package updates
	+ More than 100 new packages


 Por favor, leed el [Locris ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) for more information.  

 Download for i686: See our [lista de servidores](http://frugalware.org/download/frugalware-testing-iso). ¡No olvidéis comprobar la [integridad](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) de las imágenes de instalación antes de grabarlas!
