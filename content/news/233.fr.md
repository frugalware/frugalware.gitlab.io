L'équipe de développement de Frugalware a le plaisir de vous annoncer la sortie de Frugalware 1.7pre1, la première préversion technique de la version stable 1.7 à venir.   
  

 Voici quelques uns des changements et corrections apportés :  

* Paquets mis à jour :
	+ Kernel 3.3
	+ KDE 4.8.1
	+ LibreOffice 3.5.1
	+ Firefox 11.0, Thunderbird 11.0, Seamonkey 2.8
* Nouvelles fonctionnalités :
	+ les modules du Kernel sont maintenant compressés avec XZ.
	+ GRUB2 a remplacé GRUB1.
	+ L'installeur texte prend en charge les partitions GPT.
	+ pm-radeon est disponible, pour configurer la gestion de l'énergie de KMS Radeon lors du démarrage.
	+ Le paquet phc offre la possiblité - expérimentale - de ssous-alimenter en tension le processeur.


 Veuillez vous référer au [Changelog de -Testing](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) pour davantage d'informations.  

 Téléchargement pour x86\_64 : regardez notre [liste de miroirs](http://frugalware.org/download/frugalware-testing-iso). N'oubliez pas de vérifier [l'intégrité](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) des fichiers avant de les graver.  

**Note :**  cliquez [ici](http://frugalware.org/docs/install#_choosing_installation_flavor) pour savoir de quel média vous avez besoin pour l'installation.
