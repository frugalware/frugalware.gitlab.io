Dans quelques heures, c'est Noël, et devinez quoi ? L'équipe de développement de Frugalware a le plaisir de vous offrir un cadeau : la nouvelle version de FWLive (Frugalware Live CD/DVD).  

* FWLive est disponible dans diffèrentes versions :
	+ GNOME Desktop Live-CD (fwlive-gnome-\*.iso).
	+ KDE Desktop Live-CD (fwlive-kde-\*.iso).
	+ XFCE Desktop Live-CD (fwlive-xfce-\*.iso).
	+ LXDE Desktop Live-CD (fwlive-lxde-\*.iso).
	+ Educative Edition Live-CD (fwlive-education-\*.iso).
	+ Gamer Live-DVD Edition (fwlive-gamer-\*.iso).
* A savoir :
	+ Par défaut le clavier est en qwerty.
	+ Pour l'instant, le seules langues disponibles sont : en, de, fr, it, hu.
	+ Le login et mot de passe de l'utilisateur par défaut sont: fwlive.
	+ Le mot de passe du compte root est: fwlive.
	+ L'installateur graphique de Frugalware est ajouté par défaut.
	+ La Educative Edition est dédiée à l'apprentissage des enfants.
	+ La Gamer Edition est un DVD live plein de jeux libres.


NOTE : **Cette version est encore beta.**  

Téléchargement pour i686 et x86\_64 : consultez notre [Liste de miroirs](http://frugalware.org/download/frugalware-testing-iso). N'oubliez pas de vérifier [l'intégrité](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) des fichiers .iso avant de les graver. Les images ISO avec le suffixe -Fr sont des versions en français (et donc avec clavier en azerty).
