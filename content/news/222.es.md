El Equipo de Desarrolladores de Frugalware tiene el placer de [anunciaros](/news/222) la disponibilidad inmediata de Frugalware 1.6pre2, la segunda revisión técnica de la futura versión estable 1.6.
Aquí tenéis algunas de las mejoras, correcciones y actualizaciones más importantes desde la 1.6pre1:  

* Paquetes actualizados:
	+ Linux kernel 3.1
	+ XOrg server 1.11.2
	+ KDE SC 4.7.4
	+ Firefox 8.0.1
	+ Chromium 16.0.912.63
	+ Blender 2.60
	+ La versión 1.1.0 del componente central de las EFL
* Nuevas mejoras:
	+ Todos los servicios usados por defecto en la instalación tienen unidades nativas de systemd
	+ El instalador de las imágenes usa isohybrid en x86, ya no se darán imágenes separadas para las memorias USB.
	+ Chrony es el client NTP por defecto.
	+ /boot/grub/menu.lst y /etc/fstab contienen las UUID en las nuevas instalaciones.


 Por favor, leed el [ChangeLog de Testing](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) (en inglés) de Frugalware para más información.  


 Descargar las iso para i686: Mirad nuestra [lista de servidores](http://frugalware.org/download/frugalware-testing-iso). ¡No olvidéis comprobar la [integridad](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) de las imágenes de instalación antes de grabarlas!  

**NOTA**: Haz click [aquí](/docs/install#_choosing_installation_flavor) (en inglés) para saber más sobre qué dispositivo necesitas para la instalación.   

  

 ¡Ah, y antes de que se nos olvide, os deseamos Feliz Navidad!
