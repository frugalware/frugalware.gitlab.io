L'équipe des développeurs de Frugalware est fière de vous [annoncer](/news/107) la sortie de Frugalware 0.9rc1, notre première release candidate pour la version stable 0.9.  

 Une liste courte et incomplete des changements depuis la version 0.9pre2:  

* Améliorations:
	+ Les fichiers de documentation vide (README, NEWS, etc.) ont étés retiré de /usr/share/doc.
	+ Correction de l'installation de virtualbox.
	+ firefox-adblock a été remplacé par firefox-adblock\_plus.
	+ L'environnement ROX est maintenant disponnible aussi sur x86\_64.
	+ Plus de 800 petits changements.
* Mise à jours de paquets:
	+ Librairie GNU C 2.8
	+ Firefox 3.0.1
	+ OpenOffice.org 2.4.1
	+ Plus de 400 mise à jour de paquet.
* Nouveaux paquets:
	+ dblatex (Produits des documents DVI, PostScript, PDF à partir de sources DocBook).
	+ wxruby (extensions wxWidgets pour ruby).
	+ 10 nouveaux jeux.
	+ Plus de 70 nouveaux paquets.


 Veuillez vous référer à la Frugalware [Testing ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) pour plus d'informations.  

 Nous avons un [planing](http://bugs.frugalware.org/?do=roadmap) pour les nouvelles fonctionnalités nous prévoyons d'ajouté avant que Solaria soit stable.
