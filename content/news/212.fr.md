L'équipe de développement de Frugalware a le plaisir de vous [annoncer](/news/210) la sortie de Frugalware 1.5rc1, la première Release Candidate de la version 1.5 stable à venir.
 Vous trouverez ici quelques uns des changements apportés depuis la version 1.5pre2 :  

* Paquets mis à jour :
	+ KDE 4.6.5
	+ Mozilla Firefox 5.0
	+ LibreOffice 3.4.1
	+ FFmpeg 0.8
* Nouvelles fonctionnalités :
	+ Le paquet Python *setuptools* a été remplacé par *distribute*.
	+ Notre paquet gdb a
	  [le support pour Python](http://sourceware.org/gdb/wiki/PythonGdbTutorial).


 Veuillez vous référer au [Changelog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/Changelog.txt) pour plus d'informations.  
  

 Téléchargement pour i686 : regardez [notre liste de miroirs](http://frugalware.org/download/frugalware-testing-iso). N'oubliez pas de vérifier [l'intégrité](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) des images ISO avant de les graver !  
  

**NOTE** : cliquez [ici](/docs/install#_choosing_installation_flavor) pour savoir de quel média vous avez besoin pour installer Frugalware.
