L'équipe de développement de Frugalware a le plaisir de vous annoncer la sortie de Frugalware 1.5pre2, la seconde préversion de la version 1.5 stable à venir.Vous trouverez ici quelques uns des changements apportés depuis la version 1.5pre1 :  

* Paquets mis à jour :
	+ Kernel Linux 2.6.39
	+ LibreOffice 3.4
	+ KDE 4.6.4
* Nouvelles fonctionnalités :
	+ L'écran de démarrage est désormais fourni par [Plymouth](http://www.freedesktop.org/wiki/Software/Plymouth).
	+ Les noyaux pour les architectures i686 et x86\_64 peuvent maintenant démarrer depuis EFI (Cependant, le chargeur de démarrage n'est pas installable, pour le moment, depuis setup).
	+ En complément du support EFI, la table de partition GPT est maintenant activée. gptfdisk est l'outil recommandé pour manipuler GPT.


 Veuillez vous référer au [Changelog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/Changelog.txt) pour plus d'informations.  
  

 Téléchargement pour i686 : regardez [notre liste de miroirs](http://frugalware.org/download/frugalware-testing-iso). N'oubliez pas de vérifier [l'intégrité](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) des images ISO avant de les graver !  
  

**NOTE** : cliquez [ici](/docs/install#_choosing_installation_flavor) pour savoir de quel média vous avez besoin pour installer Frugalware.
