L'équipe de développement de Frugalware a le plaisir de vous annonce la sortie de Frugalware 1.5rc2, la seconde Release Candidate de la version 1.5 stable à venir.
 Voici une partie des modifications apportées depuis la 1.5rc1 :


* Paquets mis à jour :
	+ Linux Kernel 2.6.39.3
	+ XOrg server 1.10.3
	+ LXDE mis à jour (lxsession-lite-0.4.6, lxpanel-0.5.7, lxlauncher-0.2.2, lxterminal-0.1.10...)
	+ Version 1.0.1 des librairies EFL.
	+ Mozilla Firefox 5.0.1
	+ 245 paquets mis à jour
	+ 20 nouveaux paquets
* Nouvelles fonctionnalités :
	+ Notre paquet Vim dispose désormais du support pour Python


 Veuillez vous référer au [Changelog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/Changelog.txt) pour plus d'informations.  

 Téléchargment pour i686 : regardez notre [liste de mirroirs](http://frugalware.org/download/frugalware-testing-iso). N'oubliez pas de vérifier [l'intégrité](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) des images iso avant de les graver !  

**Note :** cliquez [ici](/docs/install#_choosing_installation_flavor) pour connaître les pré-requis nécessaires à l'installation.
