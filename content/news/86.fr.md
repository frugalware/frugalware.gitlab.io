Les développeurs de Frugalware ont le plaisir de vous [annoncer](/news/86) la disponibilité disponibilité de Frugalware 0.8rc1, la première pré-version de la prochaine version stable 0.8.  

 Une courte et incomplète liste des changement depuis la version 0.8pre2:  

* Correction de bugs:
	+ Splash amélioré: il reste maintenant présent jusqu'à la fin du processus d'arrêt de l'ordinateur.
	+ Correction de la méthode d'installation de paquet "un par un" dans l'installateur.
	+ Le LiveCD supporte désormais la copie vers la RAM si celle-ci est suffisante afin d'avoir le lecteur CD/DVD disponible.
* Mise à jour de paquets:
	+ Linux 2.6.23.14 + corrections sécurité
	+ Xorg 7.3 + corrections sécurité
	+ GNOME 2.20.3
	+ Plus de 300 mises à jour de paquets
* Nouveaux paquets:
	+ etckeeper - garder votre /etc sous contrôle de version
	+ linuxwacom - support pour les tablettes Wacom
	+ Nouveaux pilotes Xorg
	+ Plus de 20 nouveaux paquets


 Veuillez vous référer à la Frugalware [Testing ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) pour plus d'informations.
