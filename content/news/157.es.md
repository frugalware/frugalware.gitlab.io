El Equipo de Desarrolladore de Frugalware tiene el placer de [anunciar](/news/157) la disponiblidad de Frugalware 1.2pre2, the second technical preview of the upcoming 1.2 stable release.  

 A short and incomplete list of changes since 1.2pre1:  

* Improvements:
	+ Introduced devtmpfs: you can now exclude /dev from backups, along with /proc and /sys.
	+ Removed several large games from the install images (CD/DVD) which means one less DVD or 4 less CDs.
* Paquetes actualizados:
	+ Linux kernel 2.6.32 + fixes
	+ GNU C Library 2.11
	+ Mozilla Firefox 3.5.7
	+ Thunderbird 3.0
	+ More than 800 other package updates
	+ More than 100 new packages


 Por favor, leed el [Locris ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) for more information.  

 Download for i686: See our [lista de servidores](http://frugalware.org/download/frugalware-testing-iso). ¡No olvidéis comprobar la [integridad](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) de las imágenes de instalación antes de grabarlas!
