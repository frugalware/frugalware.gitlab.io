El Equipo de Desarrolladores de Frugalware se complace en [anunciarles](/news/206) la disponibilidad inmediata de Frugalware 1.5pre1, la primera candidata de la futura estable 1.5.
Aquí tenéis algunas de las mejoras, correcciones y actualizaciones más importantes de la 1.4:  

* Paquetes actualizados:
	+ Linux kernel 2.6.38
	+ Gnome 3.0
	+ KDE 4.6
	+ Firefox 4.0
	+ XFCE 4.8 con algunos bugs corregidos (4.8.1), mirad [aqui](http://www.xfce.org/about/news/?post=1295136000) (en inglés) para ver los detalles de lo que trae esta nueva versión.
	+ Disponible las librerias EFL 1.0 [Más información](http://www.enlightenment.org/p.php?p=news/show&l=en&news_id=28) (en inglés)
* Nuevas mejoras:
	+ [systemd](http://www.freedesktop.org/wiki/Software/systemd) es ahora el sistema init por defecto
	+ [Iniciado (en francés)](http://elentir.sleipnir.fr/?p=489&lang=en) el port para ARM
	+ XFCE tiene lcdfilter para fuentes tipográficas tipo GNOME
	+ El archivo completo de paquetes sólo esta disponible en DVD, en su lugar, ofrecemos una imagen pequeña de CD (~200MB) para instalar un sistema mínimo.


 Por favor, leed el [Testing ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) (en inglés) de Frugalware para más información.  

 Descargas para i686: Mirad nuestra [lista de mirror](http://frugalware.org/download/frugalware-testing-iso). ¡No olvidéis de comprobar la [integridad](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) de las imágenes de instalación antes de grabarlas!  

**NOTA**: Haz click [aquí (en inglés)](/docs/install#_choosing_installation_flavor) para saber más sobre que dispositivo necesitas para la instalación.
