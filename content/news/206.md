The Frugalware Developer Team is pleased to [announce](/news/204) the immediate availability of Frugalware 1.5pre1, the first technical preview of the upcoming 1.5 stable release.
Here are some of the major improvements, fixes and updates since 1.4:  

* Package updates:
	+ Linux kernel 2.6.38
	+ Gnome 3.0
	+ KDE 4.6
	+ Firefox 4.0
	+ XFCE 4.8 with some bug fix releases (4.8.1), see [here](http://www.xfce.org/about/news/?post=1295136000) the details brought by this version.
	+ 1.0 release of the core EFL libraries ([details](http://www.enlightenment.org/p.php?p=news/show&l=en&news_id=28))
* New features:
	+ [systemd](http://www.freedesktop.org/wiki/Software/systemd) is now the default init system
	+ [Initial](http://elentir.sleipnir.fr/?p=489&lang=en) ARM port
	+ XFCE now has lcdfilter for fonts like GNOME
	+ Full package archive is now available on DVD only, instead we offer a smaller (~200MB-sized) CD for installing a base-only system.


 Please refer to the Frugalware [Testing ChangeLog](http://ftp.frugalware.org/pub/frugalware/frugalware-testing/ChangeLog.txt) for more information.  

 Download for i686: See our [mirror list](http://frugalware.org/download/frugalware-testing-iso). Don't forget to check the [integrity](http://frugalware.org/download/frugalware-testing-iso/SHA1SUMS) of the install images before burning!
