Tips and tricks

This section relies on your contributions! If have some tips and tricks that you would like to be shown in the newsletter, please post them on the forums in the Tips and Tricks section!

  • Set your time and date correctly with ntpdate

    NTP is the Network Time Protocol. You can install it with: 
    pacman-g2 -S ntp
    Next, you need to run ntpdate (nl.pool.ntp.org is only an example, you should put a server near you here): 
    ntpdate nl.pool.ntp.org
    Finally, to have ntpdate run on boot, add this line to /etc/rc.d/rc.local: 
    ntpdate nl.pool.ntp.org &
    Of course, replace nl.pool.ntp.org with the correct server.

Security announcements

Here are a list of security issues which have been discovered and fixed in the 0.6 release in the past 2 weeks:

  • FSA209 - kernel: Two vulnerabilities and a weakness have been reported in the Linux Kernel, which can be exploited by malicious, local users to disclose potentially sensitive information and malicious people to cause a DoS (Denial of Service). Please upgrade to kernel-2.6.20-5terminus6.
  • FSA208 - mutt-ng: A vulnerability has been reported in mutt, which potentially can be exploited by malicious, local users to gain escalated privileges. Please upgrade to mutt-ng-20070125-2terminus1.
  • FSA207 - cacti: A vulnerability has been discovered in Cacti, which can be exploited by malicious people to cause a DoS (Denial of Service). Please upgrade to cacti-0.8.6j-2terminus1.
