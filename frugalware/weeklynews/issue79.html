EFI support enabled in the kernel

Support for EFI was recently enabled in the Linux kernel of 'current' by developer ryuo. He kindly agreed to provide an explanation of EFI for the newsletter and here it is:

Newer laptops, netbooks, desktops, and other systems using x86 and/or x86_64 motherboards are beginning to come with EFI/UEFI support. This support in the kernel is critical for booting. These systems cannot boot without this support. Although they have legacy support for the old MBR booting, it may eventually be unavailable.

EFI/UEFI has a master boot manager that is maintained in the hardware. This eliminates the need for using GRUB for having a multiboot setup. Under MBR, this was one of the few ways to multiboot. You had to use a specific software implementation. Under EFI/UEFI, GRUB can be reduced to booting only linux. Windows 7 64 bit has its own EFI/UEFI bootloader, so you no longer need GRUB to multiboot with it.

EFI/UEFI still works with the MS DOS partition table, but it is best to use it with GPT partition table. Windows 7 will only use EFI if GPT is in use. Switching to GPT also allows partitions to be created of beyond 2 TB. And, GPT does away with primary/extended partitions and leaves you with a flat limit of 128 partitions.

Tips and Tricks

Skeleton files

When a new user is created on Linux, the contents of the 

/etc/skel
directory are copied to the users' home page. This allows the computer's administrator to specify configuration settings and/or have folders already available when the user first logs on. For anyone who is the administrator in their home ( :D ) you might use this option, for example, to encourage everyone to store files by type in specific folders: Videos, Photos, Downloads for example. To do this you create directories of the required names in /etc/skel (as user 'root'). Note that this applies only to newly created users, not existing users.

System information script

Some time ago, Frugalware developer Devil505 and Frugalware user Aptano modified a Perl script which presents information about the GUI environment on the console. Checking for outdated packages

In the previous newsletter issue I gave a link to a web page which listed all packages for which we know there are updated versions upstream - i.e. at the packages' home site. What I didn't explain was that this link appears at the bottom of the "Packages Search" web page. It was added so that if you you looking for a later version of a package, you could quickly confirm if we knew that a later version was available. For many reasons, some packages are not updated so if you find that if a package you want upgraded is not upgraded, perhaps raise a Feature Request in the Frugalware Bug Tracker System , including a brief, reasonable explanation. Everyone on the development team is busy, so a reason why you want the package upgraded helps them prioritise your request.

Focus on Packages

Quod Libet

I'm sorry to feature another audio player after featuring Pogo in the most recent newsletter but I couldn't help myself. :) Quod Libet is a full-featured audio player developed in Python and uses the GStreamer framework. For anyone who is looking for an alternative to players such as Banshee, Rhythmbox or Amarok, Quod Libet could be just what you're looking for.

Its features include:

  • Album art;
  • Track ratings;
  • Internet radio;
  • Numerous plugins including Scrobbling support, Tray Icon, Alarm Clock and Lullaby.

Quod Libet makes it very easy to play your music in the way you want. It offers many different ways to browse or search your music and assemble a playlist. If you need to correct your music's tags you can do this easily on either individual files or several files at once. Tag editing even includes some common built-in functions including replacing underscores with spaces, removing characters which are incompatible with Windows and removing non-ASCII characters.

One quite interesting feature is its Tray Icon plugin. Of course most audio players provide system tray support so that you can continue listening to a playlist without the player's window cluttering your desktop. Quod Libet's tray icon is better than most though because, with the mouse cursor over the icon, scrolling the mouse's wheel will change volume and holding down the Shift key and scrolling the wheel will change track. As with other music players, you can also right-click on the icon and get more functions.

Even if you're currently using a music player, try Quod Libet because it offers a great range of features while still being very easy to use.
