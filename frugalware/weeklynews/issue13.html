Tips and tricks

Disclaimer - Be aware that the hints & tips provided here have NOT been tested and so come with no warranty.

  • Changing rights of files and libraries - submitted by 'bee-carl'

    Consider a situation where you have downloaded a WordPress theme and the rights of the files and libraries are incorrect. Or your hard disk has collapsed, so that you have to put a backup from a CD to your new hard disk.

    The following steps will help you correct this situation -

    1. Navigate to the required folder
      cd /to/there/where/the_files/are_located
    2. Change all directories' rights to rwxr-xr-x
      find . -type d -exec chmod 755 {} \;
    3. Change all files' rights to rw-r--r--
      find . -type f -exec chmod 644 {} \;

    • Now the rights on files and directories are again correct.

Security announcements

According to the normal support arrangements for Frugalware, when Frugalware 0.7 (Sayshell) was released, support for the previous release ended. This means that no further security or bug fixes will be released for Frugalware 0.6 (Terminus).

Here is a list of security issues which have been discovered and fixed in the 0.7 release since the previous newsletter.

  • FSA332 - firefox: A security issue has been reported in Mozilla Firefox, which can be exploited by malicious people to conduct cross-site scripting attacks. Please upgrade to firefox-2.0.0.11-1sayshell1.
  • FSA331 - pidgin: A weakness has been reported in Pidgin, which can be exploited by malicious people to cause a DoS (Denial of Service). Please upgrade to pidgin-2.2.1-2sayshell1.
