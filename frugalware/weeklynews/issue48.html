Events

Here's a selection of events which have occurred since the previous newsletter:

  • Mibbit and FreeNode

    In newsletter issue 31 I gave readers a tip which mentioned the web IRC gateway - Mibbit - at which you could chat in IRC from within your web browser. Unfortunately Mibbit no longer allows its users to access IRC channels hosted on the FreeNode network. The reason given by FreeNode for the change was that they were getting abusive traffic via the site. To fill the gap, FreeNode offer their own web gateway - which does allow access to FreeNode-hosted channels - http://webchat.freenode.net.

  • New developer - Exceed

    No, this is not news that I've cut and pasted from an old newsletter, we really do have a new developer! Welcome to Exceed, whose contributions so far have mostly been audio-related packages. It's a positive sign for the future of Frugalware that we have had several new developers in recent months. To quote a famous sci-fi phrase, may Frugalware "Live long and prosper".

  • Frugalware Linux 1.1pre2 released

    On 7 July 2009, VMiklos announced the release of Frugalware Linux 1.1pre2, the second pre-release of version 1.1 - code-name "Getorin". As I say about every pre-release, please, please download and test it if you can. Any bugs found and fixed now means the final release will be closer to being bug free.

  • We have an GSOC entrant

    Google's Summer of Code (GSOC) is a support programme sponsored by Google (surprise!). The very exciting news for Frugalware is that VMiklos has been accepted into the GSOC. He will be working on improving PHP support for SWIG. As described on SWIG's web site - "SWIG is a software development tool that connects programs written in C and C++ with a variety of high-level programming languages. SWIG is most commonly used to create high-level interpreted or compiled programming environments, user interfaces, and as a tool for testing and prototyping C/C++ software." Just in case you don't believe me, follow this link and read the news item dated 21 April 2009.

    We're not sure just what he's planning to spend the money on, but the rumour is that his long-term goal is to own a Mercedes Benz 'Kompressor'. Congratulation, VMiklos. Perhaps in the future we will get Google's support to improve Frugalware. Now I'm a little suspicious of where the 'G' in GFpm' came from! Priyank - do you have any comment?

    Note: Some cynical geeks have claimed that the GSOC is merely a beauty competition for geeks. I know for certain that this is NOT true; I've seen photos of VMiklos! :P (No, not those types of photos, just the normal photo of his face.

Looking for a package? Help is here...

pacman-g2 and GFpm both offer the option of searching for packages, by name etc, but you can do the same with Firefox. Simply install the plugin - found on the Frugalware packages page and you can then search for packages using the native Firefox method.

To conduct a search, you select Frugalware from the drop-down list of search engines, then enter your search criteria in the search box.

What's happening with project X?

For applications and projects that are of particular interest to you, it can be very interesting to know what's going on "behind the scenes". Many projects - including Frugalware Linux - use mailing lists for announcements and discussion. Some mailing lists allow only authorised people to send messages, but everyone can read their content. Others allow everybody to both send and read messages. It's sometimes hard to keep up with the number of messages posted, but it gives you an insight into how the application or project develops. It's also a method of influencing how it develops since you can comment on changes or the development of features. Please pick your favourite project and get involved - sign up to one of their mail lists.

Bisigi Project

I want to first emphasise that the Frugalware artwork team, whose work can be viewed at http://frugalware-art.org/ is excellent.

Like many Frugalware users, I enjoy using a good-looking desktop. Of course what looks good to one person may not look good to another. My preference is for a unified theme, where the appearance of GRUB, the splash screen, login manager, desktop environment's splash screen, and wallpaper are based on the same artwork. I recently discovered the Bisigi Project, which " aims to create complete and coherent themes." You can find screenshots of all the themes here. They're packaged for Ubuntu, but they're also available in tarball format, so they're easily installed on other distributions, like Frugalware. If you also like a unified theme, I encourage you to look at these themes. Even if you don't use Gnome, you may like to the wallpapers included with each theme.

Focus On Package(s)

fbterm

fbterm is a console terminal emulator which uses the Framebuffer to provide an attractive interface, including handling modern fonts and displaying an image as the background. The background image is actually set using the fbv tool - featured below. This feature was added in April 2009 and provides you with a more attractive terminal emulator.

fbv

fbv is an application which can display images at the console, using the Framebuffer. It's used by fbterm - featured above - to load an image in the console to act as the background.

Thanks to ryuo for packaging both fbterm and fbv.
