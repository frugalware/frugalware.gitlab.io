Security announcements

Here are a list of security issues which have been discovered and fixed in the 0.6 release in the past few weeks:

  • FSA223 - joomla: Cindy Chee has discovered a vulnerability in Joomla!, which can be exploited by malicious people to conduct cross-site scripting attacks. Please upgrade to joomla--1.0.12-2terminus1.
  • FSA222 - qemu: Tavis Ormandy has reported some vulnerabilities in QEMU, which can be exploited by malicious uses to bypass certain security restrictions or cause a DoS (Denial of Service). Please upgrade to qemu-0.9.0-2terminus1.
  • FSA221 - flashplugin: Critical vulnerabilities have been identified in Adobe Flash Player that could allow an attacker who successfully exploits these potential vulnerabilities to take control of the affected system. Please upgrade to flashplugin-9.0.48.0-1terminus1.
  • FSA220 - wireshark: Some vulnerabilities have been reported in Wireshark, which can be exploited by malicious people to cause a DoS (Denial of Service). Please upgrade to wireshark-0.99.6-1terminus1.
  • FSA219 - avahi: Emanuele Aina has reported a security issue in Avahi, which can be exploited by malicious, local users to cause a DoS (Denial of Service). Please upgrade to avahi-0.6.17-3terminus1.
  • FSA218 - kernel: A vulnerability has been reported in the Linux Kernel, which can be exploited by malicious, local users to cause a Denial of Service (DoS). Please upgrade to kernel-2.6.20-5terminus7.
  • FSA217 - xvidcore: Trixter Jack has reported a vulnerability in the Xvid library, which can be exploited by malicious people to compromise an application using the library. Please upgrade to xvidcore-1.1.3-1terminus1.
  • FSA216 - wordpress: Janek Vind has discovered a vulnerability in WordPress, which can be exploited by malicious people to conduct SQL injection attacks. Please upgrade to wordpress-2.2.1-1terminus1.
  • FSA215 - openoffice.org: Some vulnerabilities have been reported in OpenOffice, which can potentially be exploited by malicious people to compromise a user's system. Please upgrade to openoffice.org-2.1.0-6terminus2.
  • FSA214 - clamav: Some vulnerabilities have been reported in ClamAV, which can be exploited by malicious people to cause a DoS (Denial of Service). Please upgrade to clamav-0.90.2-1terminus2.
  • FSA213 - gd: Some vulnerabilities have been reported in the GD Graphics Library, where some have unknown impact and others can potentially be exploited to cause a DoS. Please upgrade to gd-2.0.34-2terminus2.
  • FSA212 - evolution-data-server: Philip Van Hoof has reported a vulnerability in Evolution, which potentially can be exploited by malicious people to compromise a user's system. Please upgrade to evolution-data-server-1.10.0-2terminus1.
  • FSA211 - libexif: A vulnerability has been reported in libexif, which can be exploited by malicious people to cause a DoS (Denial of Service) and potentially compromise an application using the library. Please upgrade to libexif-0.6.13-2terminus2.
  • FSA210 - vlc: Some vulnerabilities have been reported in VLC Media Player, which can be exploited by malicious people to compromise a user's system. Please upgrade to vlc-0.8.6-4terminus1.
