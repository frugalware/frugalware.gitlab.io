Events

Here's a selection of events which have occurred since the previous newsletter:

  • Dex77 helps out with Splashy

    The following message was noted in the Splashy changelog for 0.3.6 - "uses the real size of the screen and not that reported by the surface. (Thanks to DeX77 from #frugalware for helping to get this fixed)".

    Well done Dex77!

  • FUN is now Danish

    Carl Anderson recently translated the FUN (Frugalware Update Notifier) into Danish. Thanks to Carl for his contribution and all others who are also translating Frugalware text.

    Now, what's Danish for "fun"?

  • Frugalware to be the #1 games platform! :)

    As at last count, Frugalware has 63 games and related packages in the -current repository. Big names include Tremulous, Chromium, Nexuiz and Extreme Tuxracer.

    Move over Playstation 3, XBox 360 - here comes Frugalware!

  • Black(listing) is the new Black

    Blacklisting support has been improved. Users often requested to be able to blacklist for example the ipv6 module, but it was not possible until now, with udev>=118-1. Just list it in the usual `/etc/sysconfig/blacklist` file! As a side effect, autodetecting hardware is now done before everything else - silently. So don't be surprised that the `hotplug` service disappeared from the middle of the boot process.

Tips and tricks

  • Reverse SSH tunnel - submitted by 'vmiklos'

    If you have a client behind a firewall and you have a server somewhere. and you want to ssh from the server to the client, you can use ssh's magic -R option!

    if you do an

    $ ssh -R 14443:localhost:22 server

    You can ssh to client:22 from the server by ssh-ing to localhost:14443 on the server. It works even if the client is behind a firewall!
