Upcoming features

We've got a whole load of new features planned for Frugalware 0.6, and some of us are also looking forward to 0.7. Here's a short list of things we've been discussing recently:

  • Initramfs

    Initramfs is a replacement for the old initial ramdisk system that was used to provide a small system that runs from memory when the kernel starts which loads kernel modules that enable the rest of your system to boot. Frugalware didn't use initrd, which meant that some systems with hardware that has it's drivers compiled as a module may not be able to boot. Soon we'll be adding support for initramfs to Frugalware, which means Frugalware should run on a much wider range of computers, especially servers with the root file system on a SCSI drive.

  • New init system

    Quite a few of you may have heard our plans to replace SysV INIT, the program which boots the system, in either 0.6 or 0.7. Ubuntu have also done this, and we hope to use their great work, Upstart. I (AlexExtreme) am working with the Upstart developer, helping with the design, contributing a few patches, and things like that. Hopefully it'll be ready for 0.6, if not, 0.7 will have Upstart.

  • Setup improvements

    Currently our installation program is rather buggy, and has some usability problems. Also it doesn't work for unknown reasons on some machines. We're going to try our best to fix a lot of these problems for 0.6, and improve the overall stability and usability of Setup.

  • FwLive

    FwLive is a LiveCD version of Frugalware, which is currently in development. It runs completely from the CD, with GNOME, KDE and Server versions. You can also create your own custom CDs, too. We are also writing a new installation program which installs the content of the Live CD to your hard disk, in a similar fashion to the Ubuntu and Mandriva One live installers.

  • NetworkManager

    Users of wireless networks will be pleased to know that we are in the process of trying to get NetworkManager, a great networking management tool, working on Frugalware. It can detect wireless networks in your vicinity, and assists you in connecting to them. For more information, see the NetworkManager homepage
